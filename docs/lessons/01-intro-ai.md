# Introduzione all'Intelligenza Artificiale

## Cos'è l'Intelligenza Artificiale?

L'**Intelligenza Artificiale (AI)** è la capacità di una macchina di eseguire compiti che normalmente richiedono intelligenza umana: ragionare, imparare, riconoscere pattern, comprendere il linguaggio naturale.

Non si tratta di "magia": è matematica, statistica e grandi quantità di dati.

> "L'AI non sostituisce l'intelligenza umana — la amplifica."

## I tre paradigmi dell'apprendimento

### 1. Apprendimento Supervisionato
Il modello impara da esempi etichettati: **input → output corretto**.

```
Email → Spam / Non Spam
Immagine → "gatto" / "cane"
```

### 2. Apprendimento Non Supervisionato
Il modello trova pattern nei dati **senza etichette**.

Esempi: clustering di clienti, compressione di dati, rilevamento anomalie.

### 3. Apprendimento per Rinforzo
L'agente impara attraverso **premi e punizioni** interagendo con un ambiente.

Esempi: AlphaGo, robot che imparano a camminare, trading algoritmico.

## Machine Learning vs Deep Learning vs AI

```
AI (tutto)
└── Machine Learning (impara dai dati)
    └── Deep Learning (reti neurali profonde)
        └── LLM (modelli linguistici, es. Claude, GPT)
            └── Agenti AI (LLM + memoria + strumenti)
```

Il **Deep Learning** usa reti neurali con molti strati (*layer*) che imparano rappresentazioni sempre più astratte dei dati.

## Applicazioni reali oggi

- **Linguaggio**: traduzione, riassunto, generazione di testo, chatbot
- **Visione**: riconoscimento facciale, guida autonoma, diagnostica medica
- **Audio**: speech-to-text, text-to-speech, musica generativa
- **Codice**: GitHub Copilot, debugging automatico, generazione di test
- **Business**: automazione processi, analisi dati, customer support

---

## Perché ora è diverso

Tre fattori hanno accelerato l'AI:

1. **Dati** — internet ha prodotto miliardi di esempi di linguaggio, immagini, codice
2. **Compute** — GPU e TPU hanno reso possibile addestrare modelli enormi
3. **Architetture** — i Transformer (2017) hanno sbloccato capacità mai viste prima
