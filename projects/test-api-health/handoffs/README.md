# Handoffs

Contiene le consegne formali tra agenti o fasi.

## Standard

- `standards/handoff-standard.md`

## Regole

- Ogni handoff deve indicare mittente, destinatario, output, file coinvolti, decisioni, problemi aperti, rischi residui e prossima azione.
- Un handoff non sostituisce una review quando il review gate e richiesto.
