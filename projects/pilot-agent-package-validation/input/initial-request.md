# Initial Request

Creare una checklist operativa per validare un Agent Package prima dell'esecuzione con Manual Execution Adapter.

La checklist deve aiutare Pipeline Supervisor o un operatore umano a capire se un Agent Package e:

- ready;
- incomplete;
- blocked.

La checklist deve essere pratica, verificabile e coerente con:

- `standards/agent-package-standard.md`
- `runtime-adapters/manual-execution.md`
- `standards/human-gate-standard.md`
- `standards/handoff-standard.md`

Il progetto pilota deve attraversare la pipeline minima della factory e produrre almeno un deliverable e un handoff.
