# Generated Agents

Contiene gli Agent Package temporanei prodotti dal Knowledge Compiler.

## File attesi

- uno o piu file `<role>-<context>.md`

## Standard

- `standards/agent-package-standard.md`

## Regole

- Ogni Agent Package deve dichiarare `agent-source`.
- La sorgente puo essere un archetype o una definizione ad hoc nell'Execution Blueprint.
- Gli Agent Package sono temporanei e validi per questo progetto.
