# Knowledge Candidates

Contiene proposte di miglioramento nate durante il progetto.

## Standard

- `standards/knowledge-candidate-standard.md`

## Regole

- Le candidate sono proposte, non conoscenza permanente.
- Knowledge Evolution decide se accettare, respingere, integrare o deprecare.
- Ogni candidate deve indicare contesto, motivazione, rischio e generalizzabilita.
